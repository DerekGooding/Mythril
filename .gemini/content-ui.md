# UI Requirements
- Use Myra.UI built-in panels, buttons, labels, progress bars.
- Create a main UI panel as the root container.
- Top section: resource bars (gold, mana, etc.).
- Middle section: draggable cards.
- Bottom section: buttons (collect, start task).
- Implement drag-and-drop by overriding input methods like `OnTouchDown`.
- Use a custom skin/theme for styling.
- Make layout responsive via anchors and padding.
