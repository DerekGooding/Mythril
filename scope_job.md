# Job System Scope

This document outlines the detailed scope of the Job System, based on Final Fantasy Tactics.

## Job Unlocking

Jobs are unlocked by reaching certain levels in other jobs. The progression is as follows:

*   **Squire:** Starting Job
*   **Chemist:** Starting Job
*   **Knight:** Squire Level 2
*   **Archer:** Squire Level 2
*   **Priest:** Chemist Level 2
*   **Wizard:** Chemist Level 2
*   **Monk:** Knight Level 3
*   **Thief:** Archer Level 3
*   **Time Mage:** Wizard Level 3
*   **Summoner:** Time Mage Level 3
*   **Orator:** Mystic Level 3
*   **Mystic:** Priest Level 3
*   **Geomancer:** Monk Level 4
*   **Lancer:** Thief Level 4
*   **Samurai:** Knight Level 4, Monk Level 5, Lancer Level 2
*   **Ninja:** Archer Level 4, Thief Level 5, Geomancer Level 2
*   **Arithmetician:** Priest Level 5, Wizard Level 5, Time Mage Level 4, Mystic Level 4
*   **Bard:** Summoner Level 5, Orator Level 5 (Male only)
*   **Dancer:** Geomancer Level 5, Lancer Level 5 (Female only)
*   **Mime:** Squire Level 8, Chemist Level 8, Summoner Level 5, Orator Level 5, Geomancer Level 5, Lancer Level 5

## Development Tips

*   **Job Class Hierarchy:** Create a base `Job` class with common properties like `JobLevel` and `JP`. Then, create subclasses for each specific job.
*   **Ability System:** Create a flexible ability system that can handle the wide variety of abilities in the game. Consider using an interface-based approach.
*   **Stat Growth:** Implement a stat growth system that is tied to the character's job. This will require careful balancing to ensure that all jobs are viable.
*   **Data-Driven Design:** Use data files (like JSON or XML) to define job properties, abilities, and unlock requirements. This will make it easier to modify and balance the game without having to recompile the code.
